# FIFA World Cup 2026 App

Мобильное веб-приложение о Чемпионате мира по футболу 2026.

## Разделы
- 🏠 Главная — live матчи, результаты, новости
- 📅 Расписание — все матчи с фильтрами
- ⚽ Бомбардиры — топ-15 снайперов
- 🧤 Вратари — статистика голкиперов
- 🏟️ Группы — 12 групп, 48 команд
- 🏆 История — все ЧМ 1930–2022
- 📰 Инсайды — новости и аналитика

## Технологии
React 18 · Vite 5 · Tailwind CSS 3

## Запуск
```bash
npm install
npm run dev
```
